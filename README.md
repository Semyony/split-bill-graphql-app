# split-bill-graphql-project
## How to run
### Server Side
install node modules and dependencies.
```sh
# /split-bill-graphql-project
npm i
# Run server-side on localhost
npm start
```
### Client Side
```sh
# go to /split-n-chat. This is client folder 
cd .\split-n-chat\
```
install node modules and dependencies.

```sh
# /split-bill-graphql-project
npm i -g ionic
npm i
ionic serve
```
